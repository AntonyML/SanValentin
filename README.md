# SanValentin

Una aplicación web romántica de San Valentín con un juego de atrapar corazones y una página de propuesta interactiva.

## Características
- Juego de atrapar corazones para desbloquear la sorpresa.
- Página de propuesta con botones que crecen y mensajes persuasivos.
- Diseño responsivo para móviles y PC.
- Fondo animado de partículas de corazones.

## Cómo ejecutar
1. Clona el repositorio.
2. Abre `index.html` en un navegador o usa un servidor local (ej. `python -m http.server 8000`).

## Tecnologías
- HTML5, CSS3, JavaScript
- SweetAlert2 para notificaciones
